# EYE teaching

EYE teaching is a way to use LLMs for not just answers, but also explanations and independent checks—what it calls the “triad of trust.” By combining data, logic, and goals into a self-contained program, it creates results that are transparent, auditable, and reliable for real-world use.

For more details, see [A Guide to EYE teaching](https://github.com/eyereasoner/eye/blob/master/teaching/EYE-teaching.md).

Run the [EYE teaching cases](https://github.com/eyereasoner/eye/tree/master/teaching/cases) to get the
[answer, reason why and check (harness)](https://github.com/eyereasoner/eye/tree/master/teaching/cases/output)
   ```bash
   ./test
   ```
