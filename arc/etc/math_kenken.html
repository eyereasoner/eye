Math Puzzle — 5×5 KenKen-style Latin (ARC)

What this is?

This is a one-file, pure JS + JSON solver for a 5×5 KenKen-style Latin puzzle. Edit the JSON and click Run ARC.

  • Data — grid N=5 and an array of cages. A cage has an operator (= + - × ÷) and a target applied to its cells.
  • Policies — declarative constraint families: row/column all-different and cage arithmetic (kept here as documentation).
  • Answer — newly derived facts (the solved digits), with the strongest checks summary.
  • Reason Why — mathematical-English explanations for cage arithmetic and Latin constraints.
  • Check — auto-generated assertions (rows, columns, cages): … items.
  • Puzzle Solution Summary — renders the solved 5×5 grid.

Engine: domain propagation (1..N), cage-tuple filtering, Latin all-different, and DFS backtracking if needed. Works nicely up to N≈6 for typical cages.

Puzzle Details

  • Size: 5×5 (values 1..5). Latin rule: each row/column contains each number exactly once.
  • Operators: = (single cell), + (sum), - (absolute diff, two cells), × (product), ÷ (integer division, two cells, no remainder).
  • Difficulty: Medium — a mix of singletons, sum cages, and a few difference/paired cages.
  • Editing: You can alter cages, targets, or size (2..6). Make sure every cell is covered by exactly one cage.
  • Uniqueness: This demo finds a solution; uniqueness is not guaranteed. To tighten, reduce singletons and add mixed operators that couple rows & columns.
  • Tip: Start with = cages to seed values; then use row/column all-different to prune domains; cage arithmetic does the rest.
